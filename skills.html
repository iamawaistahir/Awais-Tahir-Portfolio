2019 — Foundation
Finance & Bookkeeping Fundamentals
Started with core bookkeeping and accounting principles — double-entry, financial statements, and Excel financial modelling. First client engagements handling SME bookkeeping and monthly management accounts. This year established the financial discipline that underpins everything I do.
BookkeepingExcelQuickBooksFinancial Statements
2020 — Analytics Begins
SQL & Data Analysis Entry
Began learning SQL to query client databases directly — seeing that financial analysis was limited by manual Excel exports. First SQL project extracted customer revenue patterns from a 500K-row database. Immediately saw the power of structured querying over spreadsheet work. Also began advanced Excel — Power Query and pivot modelling.
SQLPostgreSQLPower QueryAdvanced Excel
2021 — Python & Power BI
Data Science Stack & Dashboard Development
Picked up Python seriously — Pandas, NumPy, and data visualisation with Matplotlib. Simultaneously learned Power BI from scratch and delivered first executive dashboard for a retail client. The combination of Python for data prep and Power BI for presentation became a core service offering. Also enrolled in ACCA qualification.

Built first full Python automation pipeline — a complete invoice generation and reconciliation system saving a client 25 hours per month. Deepened IFRS knowledge through ACCA study, applying IFRS 16 (leases) and IAS 16 (fixed assets) to client financial statements for the first time. Learned Tableau alongside Power BI.
